Section Direction:

Directions are at the top left of each test screen,

Testing Timer:

Use the timer to pace yourself. You can hide it until 5 minutes remain.

Annotate:

Select some tex, then click Annotate to highlight text or leave yourself a note.

Line Reader:

Use the line reader to help you focus as you are reading test content.

Calculator and Reference sheet:

If you are taking a digital SAT or PSAT-related assessment, you'll have access to a calculator and a reference sheet.

Help:

Click on More to access to Help content.

Expand Left/Right:

The center line and buttons let you expand either side of the screen.

Zoom: Zoom with keyboard shortcuts on laptops or by pinching on tablets. Quick tip: Zoom using control +/-(PC) or Command +/-(MAC).

Mark for review:

use the bookmark icon to flag any question you want to come back to.

Option Eliminator:

On multiple choice questions. If you think an answer option is wrong, you can cross it out.

Question Menu:

Navigate to any question in the section.
